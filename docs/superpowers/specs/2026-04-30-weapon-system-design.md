# 武器切换系统设计

## 概述
玩家每击杀5个BOSS自动切换下一把武器，共10把独特武器，全部自动攻击无需按键。

## 核心机制
- **切换条件**：击杀5个BOSS
- **切换提示**：屏幕中央大字显示武器名 + 闪光特效 + 音效
- **Power加成**：跨武器保留，Power仍然影响火力强度

## 各武器详细规格

### 1. 散弹三连
- **类型**：基础散射
- **CD**：无（持续发射）
- **Fire Rate**：与原射击频率一致
- **效果**：玩家位置发射3发子弹（1向前 + 2左右各15°）
- **Power>2**：扩展到5发（1向前 + 2左右各15° + 2左右各30°）
- **子弹外观**：青色光弹

### 2. 追踪火箭
- **类型**：特殊弹
- **CD**：2秒
- **效果**：发射1发火箭弹，曲线飞向最近的Boss，尾部粒子特效
- **寻敌逻辑**：每帧检测距离最近的Boss，火箭转向目标
- **命中**：接触Boss造成伤害后消失
- **Power>2**：伤害+20%
- **子弹外观**：红色火箭形状，尾部火焰拖尾

### 3. 回旋镖
- **类型**：轨迹弹
- **CD**：1.5秒
- **效果**：从左上方发射，逆时针绕场一圈从右下返回
- **轨迹**：贝塞尔曲线，从玩家位置→左上角→场中心→右下角→玩家位置
- **命中**：接触Boss造成伤害
- **Power>2**：轨迹半径延长
- **子弹外观**：旋转的星形，拖尾渐变消失

### 4. 能量脉冲
- **类型**：范围攻击
- **CD**：2秒
- **效果**：从玩家位置周期性释放环形能量波
- **波纹**：向外扩展的圆环，穿透所有敌弹
- **范围**：全屏
- **Power>2**：范围+15%
- **子弹外观**：青色圆环，向外扩展渐变透明

### 5. 螺旋弹幕
- **类型**：几何弹幕
- **CD**：无（持续发射）
- **效果**：发射的子弹螺旋排列向前
- **排列**：每帧发射的子弹位置按角度旋转偏移
- **螺旋速度**：每秒旋转45°
- **Power>2**：同时双螺旋（内外圈）
- **子弹外观**：多彩光弹，颜色渐变

### 6. 闪电环
- **类型**：电弧攻击
- **CD**：3秒
- **效果**：从玩家位置发出电弧环
- **外观**：锯齿状电弧，同心圆扩散
- **伤害**：全屏Boss伤害
- **消除敌弹**：电弧经过的敌弹消失
- **Power>2**：范围+15%
- **子弹外观**：白色锯齿电弧，带闪光效果

### 7. 波动炮
- **类型**：蓄力攻击
- **CD**：4秒
- **效果**：聚能0.5秒后释放球形冲击波
- **聚能提示**：玩家周围光圈渐亮
- **冲击波**：圆形波纹，穿透全场，消灭所有敌弹
- **伤害**：接触Boss造成高伤害
- **Power>2**：冲击波半径增大
- **子弹外观**：中心白色向外蓝色渐变圆环

### 8. 冰晶刺
- **类型**：控制攻击
- **CD**：2秒
- **效果**：发射冰锥，命中Boss时冻住
- **伤害**：普通子弹伤害
- **冻结**：Boss停止移动0.5秒（同效果不叠加）
- **冻结冷却**：全局5秒，防止过快刷新
- **Power>2**：冻结时间+0.2秒（最长0.7秒）
- **子弹外观**：冰蓝色尖刺，带冰晶粒子

### 9. 黑洞弹
- **类型**：引力弹
- **CD**：3秒
- **效果**：发射后原地旋转吸引敌弹，2秒后爆炸
- **吸引**：半径100px范围内的敌弹向黑洞移动
- **爆炸**：吸引结束后爆炸，消灭范围内敌弹
- **Power>2**：吸引时间+0.5秒（最长2.5秒）
- **子弹外观**：深紫色漩涡，中心发光

### 10. 时间钟摆
- **类型**：轨迹弹幕
- **CD**：2秒
- **效果**：钟摆形子弹左右横穿全场
- **轨迹**：从玩家位置发出，左右摆动穿越全场
- **摆动幅度**：30°，每秒摆动2次
- **命中**：接触Boss造成伤害
- **Power>2**：摆动幅度+5°
- **子弹外观**：金色钟摆形状，带残影

## 武器切换流程

1. 玩家击杀BOSS，数量+1
2. 累计击杀达到5时：
   - 当前武器切换完成标记
   - 武器索引+1（循环）
   - 清空杀敌计数
3. 切换动画：
   - 屏幕中央大字体显示武器名（持续1.5秒）
   - 全屏白色闪光（渐变消失，0.3秒）
   - "叮"音效播放
4. 新武器激活：
   - CD从0开始冷却
   - 子弹颜色/外观变化

## 武器显示HUD

- 左上角显示当前武器图标和名称
- 如果有CD，显示圆形冷却进度

## 保留系统

- **Power系统**：不重置，跨武器保留
- **Bomb系统**：不受影响
- **Graze系统**：不受影响
- **杀敌计数**：每武器独立，重置

## 技术实现要点

1. 新增 `weapons[]` 数组存储10个武器配置
2. `currentWeapon` 索引变量
3. `killCount` / `killsNeeded = 5`
4. 各武器独立的 `cooldown` / `lastFire` 计时
5. 武器切换时调用 `switchWeapon()` 函数
