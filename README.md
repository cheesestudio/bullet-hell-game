单文件 HTML5 Canvas 游戏，无需安装，浏览器直接打开即玩。

## 武器系统

每击败 3 个 Boss 自动切换武器。武器随 **能量等级 (0-4)** 增强——等级越高，弹幕越多、效果越强。

### 1. 散射 (Spread Shot)
- **冷却**: 无（持续射击）
- **机制**: 能量 0 单发，1 双发，2 三发，3+ 五路散射（-30°/-15°/0°/+15°/+30°）
- **特点**: 最基础的武器，覆盖面广，弹速随能量提升

### 2. 追踪火箭 (Homing Rocket)
- **冷却**: 2 秒
- **机制**: 自动追踪 Boss，能量 0 发射 1 枚，能量 3+ 最多 5 枚
- **特点**: 附带火箭尾迹特效，必定锁定 Boss 方向

### 3. 回旋镖 (Boomerang)
- **冷却**: 1.5 秒
- **机制**: 弧线飞行，利用正弦波数学形成 Lissajous 曲线轨迹，2 秒后消失
- **特点**: 最多 5 枚，攻击路径难以预测，适合压制

### 4. 能量脉冲 (Energy Pulse)
- **冷却**: 2 秒
- **机制**: 释放扩散环形 AoE，扩散速度 400px/s，最大半径 500 + 能量×50
- **特点**: 能量 3+ 可同时释放 4 层环（间隔 100ms），全屏覆盖

### 5. 螺旋弹幕 (Spiral Barrage)
- **冷却**: 无（持续射击）
- **机制**: 旋转螺旋臂，每发旋转 25+能量×3 度，最多 4 臂
- **特点**: 彩虹 HSL 色彩循环，视觉华丽，弹速较慢但覆盖密集

### 6. 闪电环 (Lightning Ring)
- **冷却**: 3 秒
- **机制**: 扩散环 350px/s，环边缘每 50ms 生成闪电粒子，最大半径 400 + 能量×50
- **特点**: 最多 3 层环，视觉效果震撼，兼顾范围伤害

### 7. 波动炮 (Wave Cannon)
- **冷却**: 4 秒（需蓄力 0.5 秒）
- **机制**: 按住释放，环形扩散 500px/s，最大半径 500 + 能量×80
- **特点**: 能量 2+ 两侧生成小型副波（半径 300+能量×40），三管齐下

### 8. 冰刺 (Ice Spike)
- **冷却**: 2 秒
- **机制**: 命中后冻结 Boss，冰冻时间 0.5 + 能量×0.1 秒，弹速 450px/s
- **特点**: 最多 5 枚，**唯一能冻结 Boss 的武器**，Boss 完全停止攻击和移动

### 9. 黑洞 (Black Hole)
- **冷却**: 3 秒
- **机制**: 生成引力场（半径 100+能量×20），吸入敌方子弹；持续 2.0+能量×0.3 秒后爆炸
- **特点**: 能量 2+ 生成第二个小型黑洞，**防御型武器**，可清除弹幕

### 10. 时间摆锤 (Time Pendulum)
- **冷却**: 2 秒
- **机制**: 摆锤式弹道，振幅 30+能量×3，以 250°/s 正弦摆动，持续 1.5 秒
- **特点**: 最多 5 枚，弹道优美，覆盖不规则区域

## Boss 系统

### Boss 名单（12 个）

`VOID` · `PULSE` · `CORE` · `NOVA` · `PHANTOM` · `SPECTER` · `CASCADE` · `VORTEX` · `ECHO` · `STORM` · `DRIFT` · `SURGE`

Boss 循环出现，每次循环血量递增，难度无上限。

### 难度递进

| 属性 | 公式 |
|------|------|
| Boss 血量 | `(200 + 波数×100) × (1 + floor(波数/3) × 0.3)` |
| 技能冷却 | `max(0.15, 0.5 - 波数×0.02)` |
| 移动速度 | `1 + 波数×0.05` |
| 技能数量 | `floor(波数/10 × 10) + 5` |

### Boss 外观

Boss 拥有多层渲染（9 层）：
1. 粒子场（20 个轨道粒子）
2. 外层能量场（径向渐变）
3. 主体（随弹幕类型变化：多边形 / 螺旋臂 / 有机体 / 钻石形 / 十字形等）
4. 内核（几何图案）
5. **眼睛**——瞳孔追踪最近玩家，血量低于 30% 时表情变为愤怒（眉毛 + 牙齿）
6. 发光核心（脉冲动画）
7. 旋转轨道环
8. 能量尖刺
9. 浮动碎片/卫星

- 前期偏向分数/炸弹，后期偏向能量/护盾/生命
- 道具 80px 范围内自动吸附到玩家
- 每个道具 8 秒后消失

## Radmin VPN 联机

**零配置，不需要端口映射，不需要公网 IP。**

最大支持 **4 名玩家** 同时联机（1 主机 + 3 客户端）。

### 网络架构

- 基于 WebSocket 中继服务器 (`relay.js`, 端口 8007)
- 第一个连接的玩家成为主机
- 固定 1/60 秒时间步长 + 种子化 PRNG (mulberry32) 确保确定性同步
- 输入以 5 位位掩码传输（左/右/上/下/炸弹）

### 前置条件

- [Radmin VPN](https://www.radmin-vpn.com/) 已安装，所有玩家在同一网络
- [Node.js](https://nodejs.org/) 已安装（主机需要）
- 运行 `npm install` 安装依赖

### 主机

```bash
node relay.js
```

看到 `Relay 中继服务器已启动 ws://0.0.0.0:8007` 即表示成功。

然后打开游戏 -> **网络对战** -> **创建房间**，把你的 Radmin VPN IP 告诉其他玩家。

等其他玩家加入后，按 **Enter** 开始游戏。

### 客户端

打开游戏 -> **网络对战** -> **加入房间** -> 输入主机 IP -> **Enter** 连接。

连接后按 **R** 准备就绪，等待主机开始。

### 大厅按键

| 操作 | 按键 |
|------|------|
| 菜单导航 | 上/下方向键 |
| 确认 | Enter / 空格 |
| 输入IP | 数字键 + 小数点，Backspace 删除 |
| 准备（客户端） | R |
| 返回 | ESC |

### 故障排查

| 问题 | 解决方案 |
|------|----------|
| relay.js 启动失败 | 确认 `npm install` 已执行，端口 8007 未被占用 |
| 连接超时 | 检查 Radmin VPN 是否同一网络，能否 ping 通主机 |
| 连接后无响应 | 检查防火墙是否允许 Node.js 网络访问 |
| 游戏不同步 | 确保所有玩家使用相同版本的 bullet-hell.html |

> 详细说明参见 [RADMIN-HOWTO.md](RADMIN-HOWTO.md)

## DG-LAB 郊狼联动

支持两种连接方式，提供触觉反馈体验。

### 方式一：内置 WebSocket

游戏内置连接 `ws://192.168.10.104:5679`，自动重连间隔 3 秒。

| 事件 | 震动 |
|------|------|
| 玩家受击 | A+B 双通道波形 |
| 击中 Boss | 轻微震动（强度 20 + 连击×2，20% 概率） |
| Boss 死亡 | 强力震动（强度 100） |

### 方式二：COYOTE SOCKET（设置中开启）

独立 WebSocket 连接，支持精细强度控制。

| 事件 | 通道1 | 通道2 |
|------|-------|-------|
| 玩家受击 | +15 | +8 |
| 拾取道具 | +5 | +3 |
| Boss 警告 | +10 | +7 |
| Game Over | 交替脉冲×10 → 双通道满载(200) |

- 全局强度乘数可调（0-100%）
- 30ms 最小脉冲间隔

| 文件 | 功能 |
|------|------|

| `game_link_server.py` | 精简版服务器 |

## 技术特性

- **确定性同步**: mulberry32 种子化 PRNG，网络模式下所有客户端生成相同弹幕
- **固定时间步**: 网络模式 1/60s 固定 tick，最大 3 步追赶
- **输入编码**: 5 位位掩码网络传输
- **弹幕上限**: 200 发硬编码限制，防止性能问题
- **Delta 限制**: 最大 0.05s，防止延迟尖峰导致瞬移
- **本地存储**: 高分和排行榜 localStorage 持久化
- **触控支持**: 移动端触控操作（触摸移动，双击炸弹）
- **响应式画布**: CSS max-width/max-height + 像素化渲染
